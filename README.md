## Konto anwendung für Schule
Author: Nadine Preiss
LBS Eibiswald 2aAPC

In dieser Anwendung habe ich eine Kontoverwaltung erstellt.
Ich kann:
•	Konto anlegen
•	Konto auflösen
•	Einzahlen
•	Abheben
•	Kontoauszug ausgeben
•	überweisen
in diesem Program machen
